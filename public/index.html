Apa itu API?

API (Application Programming Interface) adalah jembatan komunikasi antar sistem. Dengan API, aplikasi bisa saling terhubung dan bertukar data dengan mudah tanpa harus tahu bagaimana sistem lain bekerja secara internal.

Authentication

Untuk mengakses endpoint tertentu, Anda harus menggunakan API Key yang dikirim melalui header Authorization: Bearer YOUR_API_KEY.
